##################################################
VS Code に拡張機能をインストール
##################################################
VS Code に ”Git History" をインストールします。

#. VS Code を起動

   .. image:: img/2021-07-17_11h25_26.png
      :scale: 70%

#. :guilabel:`拡張機能` をクリック

   .. image:: img/2021-07-17_11h25_57.png
      :scale: 70%

#. [Marketplace で拡張機能を検索する] に "git history" と入力

   .. image:: img/2021-07-17_11h26_51.png
      :scale: 70%

#. "Git History" をクリック

   .. image:: img/2021-07-17_11h27_26.png
      :scale: 70%

#. :guilabel:`インストール` をクリック

   .. image:: img/2021-07-17_11h28_14.png
      :scale: 70%

#. インストール終了

   .. image:: img/2021-07-17_11h28_40.png
      :scale: 70%
