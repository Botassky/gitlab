アバターの変更方法¶

デフォルトのアバターの画像はちょっと複雑な模様です。この画像をサイズ 200KB 以下の画像に変更できます。

変更手順¶

  1. 右上のアイコンをクリック


  2. Settings をクリック


  3. Choose file… をクリック


  4. 画像ファイルを選択　→　開く(O) をクリック


  5. 画像の位置やサイズを調整　→　Set new profile picture をクリック


  6. アバターが選択した画像に変更されたことを確認


  7. 下までスクロール　→　Update profile settings をクリック


  8. “Profile was successfully updated” を確認　→　変更終了
