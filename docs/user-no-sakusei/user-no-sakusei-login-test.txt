.. _user-no-sakusei-login-test-kanri-sya:

管理者が作成
--------------------------------------------------
#. アカウントと仮パスワードを入力　→　:guilabel:`Sign in` をクリック

   .. image:: ./img/2020-07-01_21h31_57.png

   |

#. 各値を入力　→　:guilabel:`Create user` をクリック

   :guilabel:`New password` と :guilabel:`Password confirmation` に入力した値が本パスワードになります。

   .. csv-table::
   
      "項目", "値"
      "Current password", "hanako@password"
      "New password", "hanako@passwd"
      "Password confirmation", "hanako@passwd"

   .. image:: ./img/2020-07-01_21h32_44.png

   |

#. アカウントと本パスワードを入力　→　:guilabel:`Sign in` をクリック

   .. image:: ./img/2020-07-01_21h33_01.png

   |

#. ログイン終了

   .. image:: ./img/2020-07-01_21h33_07.png

   |
