ユーザーアカウントの作成¶

ユーザーアカウントは次のいずれかの方法で作成します。

  • ユーザー自身が作成する

  • 管理者（ root ）が作成する

● ユーザー自身が作成
● 管理者（ root ）が作成
● サインインテスト

ユーザー自身が作成¶

  1. ブラウザーで GitLab サーバーにアクセス　→　Register をクリック


  2. 各値を入力　→　Register をクリック

    次の内容でアカウントを作成します。 Username の入力後に緑色で “Username is available.” が表示されたら使用可能なユーザー名です。

    項目

    入力値

    備考

    Full name

    staff taro

    ユーザーのフルネーム

    Username

    taro

    ログインアカウント

    Email

    taro@email.local

    Email confirmation

    taro@email.local

    “Email” と同じ値

    Password

    taro@password

    アカウントのパスワード

  3. アカウントの作成終了

    Register のクリック後にユーザーアカウントが作成され、そのままサインインします。



管理者（ root ）が作成¶

管理者がユーザーアカウントを作成します。

  1. GitLab サーバーに root アカウントでサインイン


  2. Add people をクリック


  3. 各値を入力

    次の内容でアカウントを作成します。

    項目

    入力値

    備考

    Name

    staff jiro

    ユーザーのフルネーム

    Username

    jiro

    ログインアカウント

    Email

    jiro@email.local

  4. 画面をスクロール　→　 Create user をクリック


  5. “User was successfully created.” のメッセージを確認


  6. 画面右上の Edit をクリック


  7. 各値を入力　→　Save changes をクリック

    作成したアカウントのパスワードを設定します。ここで設定したパスワードは初期パスワードです。このアカウントで初めてログインするとき、本パスワードに変更します。

    項目

    入力値

    備考

    Password

    jiro@passwd

    ユーザーの初期パスワード

    Password confirmation

    jiro@passwd

    “Password” と同じ値を入力

  8. 画面をスクロール　→　 Save changes をクリック


  9. “User was successfully updated.” のメッセージを確認　→　アカウントの作成終了



サインインテスト¶

作成したアカウントで GitLab サーバーにサインインできるかテストします。

ユーザー自身が作成したアカウント¶

  1. 次の値を入力　→　Sign in をクリック

    項目

    入力値

    備考

    Username or email

    taro

    Password

    taro@password

  2. サインイン終了


管理者が作成したアカウント¶

管理者が作成したアカウントの場合、次の順序で作業を行います。

  • 本パスワードの設定

  • ログインテスト

  1. 次の値を入力　→　Sign in をクリック

    項目

    入力値

    備考

    Username or email

    jiro

    Password

    jiro@passwd

    初期パスワード

  2. 次の値を入力　→　Set new password をクリック

    項目

    入力値

    備考

    Current password

    jiro@passwd

    仮パスワード

    New password

    jiro@password

    本パスワード

    Password confirmation

    jiro@password

    “New password” と同じ値

  3. “Password successfully changed” の表示を確認


  4. 次の値を入力　→　Sign in をクリック

    項目

    入力値

    備考

    Username or email

    jiro

    Password

    jiro@password

    本パスワード

  5. サインイン終了
