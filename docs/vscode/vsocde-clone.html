クローン¶

GitLab サーバー上のリモートリポジトリをクローンし、ローカルリポジトリを作成します。

GitLab サーバー¶

  1. GitLab サーバーにログイン

  2. クローン対象のプロジェクトをクリック

  3. Clone をクリック

  4. [Clone with SSH] の右横のアイコンをクリック（ アドレスをクリップボードにコピー）

ローカルリポジトリ用のディレクトリを作成¶

ローカルリポジトリを登録するディレクトリを作成します。ディレクトリ内は空です。

クローン¶

  1. VS Code を起動

  2. リポジトリの複製 をクリック

  3. [リポジトリ URL を指定するか、リポジトリソースを選択します。] に「 :ref:`` 」でクリップボードにコピーした URL を貼り付け

    ↓

  4. クローン先のディレクトリを選択

  5. リポジトリの場所を選択 をクリック

  6. クローン終了

  7. フォルダーを開く をクリック

  8. ローカルリポジトリのディレクトリを開く

  9. OK をクリック

  10. はい、作成者を信頼します をクリック

  11. クローンしたファイル（やフォルダー）を表示

クローン後のローカルリポジトリ用ディレクトリの状態¶

クローン後のディレクトリの状態です。クローンしたリポジトリ用のディレクトリ test-project が作成されました。

test-project ディレクトリ内の状態です。 .git ディレクトリがクローンしたリポジトリの本体です。 README.md はリポジトリに含まれるファイルです。

ログを確認¶

  1. 表示 をクリック

  2. コマンドパレット… をクリック

  3. “gitlog” と入力

    ↓

  4. Git:View History (git log) をクリック

  5. ログを表示
