作業用ブランチに切り替え¶

作業用に main ブランチから dev ブランチに切り替えます。

新しいブランチを作成して切り替え¶

  1. 現在選択しているブランチを確認　→　main ブランチ

  2. ブランチ名をクリック

  3. ＋新しい分岐の作成… をクリック

  4. [ブランチ名] にブランチ名を入力

    ↓

  5. ブランチの切り替え終了

  6. dev ブランチに切り替え後のログの状態です。

既存のブランチに切り替え¶

  1. 現在選択しているブランチを確認　→　main ブランチ

  2. ブランチ名をクリック

  3. dev をクリック

  4. ブランチの切り替え終了
