Git をインストール¶

VS Code で GitLab を操作するには Git がインストールされていなければなりません。

実行コマンド¶

Git がインストール済みか確認し、未インストールであればインストールします。

whereis git
git --version
sudo apt -y install git

実行ログ¶

実行ログです。今回は Git が未インストールだったのでインストールしました。

piyo@vmpc:~$ whereis git
git:
piyo@vmpc:~$ git --version

コマンド 'git' が見つかりません。次の方法でインストールできます:

sudo apt install git
